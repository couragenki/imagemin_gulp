# imagemin_gulp

コピペだけで簡単に導入できる画像圧縮・変換機能を実装しました。

基本的にいろんなリポジトリで submodule として使う用なのでかなりシンプルにしています。

## 使い方

1.  **セットアップ**

    - `npm install` を実行します。
    - (ローカルに gulp がインストールされるため、グローバルインストール `npm i gulp -g` は不要です)

2.  **画像を入れる**

    - `src` フォルダに最適化したい画像を入れます。
    - サブフォルダに入れても構いません。

3.  **タスクの実行**

    - **画像圧縮:** `npx gulp img`
    - **WebP 変換:** `npx gulp webp`
    - **リサイズ & リネーム:** `npx gulp resize`
    - **画像圧縮と WebP 変換を同時に実行:** `npx gulp`

4.  **画像の出力**
    - **圧縮画像 & WebP 画像:** `dist` フォルダに、元のファイル構造を維持して出力されます。
    - **リサイズ & リネーム画像:** `dist/resized` フォルダに `[元のファイル名]_[幅]x[高さ].拡張子` という名前で出力されます。
